ğŸ•¹ï¸ Gameplay

Start: The game selects a random animal name from its internal list.

Guessing: Click on the on-screen keyboard, or use your physical A-Z keys, to guess a letter.

Correct Guess: If the letter is in the word, it will appear in the blanks, and the corresponding key will turn green.

Incorrect Guess: If the letter is not in the word, the hangman image will advance one stage, and the key will turn red. You have 6 lives (stages).

Win/Loss:

Win: Guess all letters before the hangman is complete.

Loss: The hangman is completed (0 lives remaining).

Restart: A game over screen will appear, allowing you to quickly start a new game.
